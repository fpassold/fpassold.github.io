O que é Codesys?

O Codesys é um ambiente de programação para PLCs e é utilizado por muitas empresas de automação industrial como Bosch, Beckhoff, Festo, Eaton, Wago e muito mais. Você pode programar usando a lógica de escada tradicional para o texto estruturado mais moderno e gráfico de fluxo contínuo (padrões IEC 61131-3).

O que é o "Codesys Control for Raspberry"?

O CodesyS Control for Raspberry Pi SL contém um aplicativo codesys control para todos raspberry Pi, incluindo o Módulo computacional, bem como a capacidade de usar os módulos de extensão e vários dispositivos/breakouts com SPI, I²C ou interface de comunicação de 1 fio.

Este produto pode ser instalado com o plug-in incluído codesys deploy tool através do CodeSYS Development System no sistema Linux do Raspberry Pi.

Obs.: Após cada reinicialização, o sistema de tempo de execução será iniciado automaticamente. Se nenhuma licença completa válida puder ser encontrada, o CONTROLE CODESYS será executado por duas horas sem limitações funcionais antes de ser desligado.

Link do fabricante: [CODESYS Control for Raspberry Pi SL].

Atenção: você deve se cadastrar neste site para poder ter acesso a uma versão free ("SL") cuja run-time library rodará continuamente por 2 horas no Raspberry.

Interfaces previstas:

The CODESYS device description does support the following components:

  • Codesys OPC UA Server, como versão completa para troca de dados
  • CANopen via EL6751 Gateway
  • Câmera Raspberry Pi
  • Interface I²C, com SenseHat, SRF02, Adafruit PWM, MPU6050 Gyro, MPU9150 Gyro, AK8975 Compass
  • Interface SPI, com MCP3008, MCP23S17, PiFace Digital, PiFace Control Display
  • Interface de um fio, com DS20B18
  • GPIO

Visualização

CODESYS WebVisu, está incluído como versão completa na entrega do pacote de runtime.

Extensão SL

O Pacote de Extensão SL está incluído no pacote de runtime e oferece funções adicionais:

  • Integração de código C existente
  • Implementação de funções externas
  • Suporte de switches start/stop
  • Uso de I/Os locais
  • Uso de tarefas de eventos externos
  • Conecte memórias persistentes (Retenções

Suporte a fieldbus

  • Mestre CODESYS EtherCAT
  • CODESYS EtherNet/IP Scanner / Adaptador
  • CODESYS J1939
  • CODESYS Modbus TCP Master / Escravo
  • CODESYS Modbus Serial Master / Slave
  • Controlador profinet /dispositivo codesys profinet

Requisitos

  • Plataformas/dispositivos suportados

    • Todos os modelos Pi Nota: Use a ferramenta "Leitor de Dispositivos" para descobrir os recursos suportados do seu dispositivo (componente gratuito do CodeSYS Development System).

  • Requisitos adicionais

    • O Controle CODESYS requer uma interface de rede (LAN, WLAN), especialmente com o Módulo De computação Raspberry Pi e o Raspberry Pi ZERO.

    • Se o produto for utilizado para fins industriais, é responsabilidade do fabricante do sistema garantir o cumprimento das especificações necessárias.

    • O sistema operacional Raspberry Pi OS (32 bits), disponível à partir de: https://www.raspberrypi.org/downloads/

    • Bibliotecas dinâmicas necessárias pelo binário codesys control runtime:

      • libm.so.6
      • libpthread.so.0
      • libdl.so.2
      • librt.so.1
      • libc.so.6
      • libgcc_s.so.1

  • Acessórios obrigatórios

    • Cartão SD (mínimo de 4GB)
    • Opcional: Chave CODESYS

Baixar arquivos à partir de: [CODESYS Control for Raspberry Pi SL]. Arquivo: CODESYS Control for Raspberry PI 4.5.0.0.package (35,3 Mbytes; versão de 30.06.2022) Obs.: Esta extensão não é reconhecida nem pelo Raspbery nem pelo Windows.

Para baixar CoDeSys (COntroller DEvelopment SYStem) acesse o seguinte link:

https://www.codesys.com/download

ou vá diretamente para a loja online do Codesys:

Antes de realmente baixar o programa, você deve primeiro obter algumas credenciais de login. Clique no link "Registro" (em vermelho) e, em seguida, você terá que selecionar um país e preencher um formulário. Em seguida, você receberá um e-mail com algumas credenciais de login.

Se você retornar à central de download (onde você clicou no link de registro vermelho), agora você pode baixar o arquivo exe para o codesys V3 mais recente depois de digitar o nome de usuário e senha.

Uma vez que o arquivo tenha sido baixado, abra o arquivo exe e se o windows tentar bloqueá-lo selecione "executar de qualquer maneira". Siga o assistente de instalação. Notei ocasionalmente que ao baixar isso você recebe um erro e o assistente de instalação não vai iniciar, se isso acontecer, tente baixar novamente o arquivo exe.

Se você está lutando por espaço, você só precisa instalar os seguintes recursos:

  • CODESYS V3;
  • CODESYS Gateway;
  • CODESYS OPC Sever;

Depois de instalar o CODESYS, você precisa adicionar o Raspberry Pi Package a ele. Para fazer isso, primeiro você tem que visitar a loja CODESYS e baixar o pacote. A loja estabelece um preço de 35 euros, no entanto, isso é apenas se você deseja executar o seu programa no Pi por mais de 2 horas. Caso contrário, é completamente gratuito e você só precisa baixar novamente o programa para o Pi a cada 2 horas, afinal CODESYS é direcionado para a indústria...

http://store.codesys.com/codesys-control-for-raspberry-pi-sl.html

Infelizmente para baixar este pacote você precisará de outro nome de usuário e senha, então você terá que se cadastrar na CodesYS Store. Se você clicar no botão de download, ele solicitará que você faça login, se você seguir este link você pode se cadastrar na loja. Depois de se inscrever, retorne à página de download e clique no download. Baixe este em algum lugar memorável, você vai precisar dele na próxima etapa.

Uma vez que o pacote tenha sido baixado, você precisará abrir codesys e instalá-lo. Vou explicar como fazer isso no próximo passo.
