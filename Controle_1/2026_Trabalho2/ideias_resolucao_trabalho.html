Idéias para Resolução do Trabalho

Sugere-se criar diferentes rotinas capazes de processar dados contidos em arquivos do tipo .csv (que podem ser editados à mão, em planilhas eletrônicas, etc).

Note que o trabalho exige uma sequência de operações:

  1. Cálculo dos componentes de uma onda, envolvendo: harmônica, frequência, amplitude e fase, gerando uma matriz ou tabela de dados da "componentes originais".
  2. Calcular a atenuação e defasagem introduzida em componente frequencial. Isto significa ler os dados da matriz ou tabela de dados da "componentes originais" e com base na função transferência do filtro, calcular nova matriz ou tabela dados "componentes originais".
  3. Sintetizar um vetor temporal baseado nos componentes espectrais das mesma, usando a matriz ou tabela de dados "componentes originais" para gerar/mostrar a "onda original" e usar a mesma rotina mas usando dados da matriz ou tabela de dados "componentes filtrados" para gerar/mostrar a "onda filtada".
