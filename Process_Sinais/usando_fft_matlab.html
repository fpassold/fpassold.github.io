Seja o seguinte exemplo:

Podemos criar um vetor de frequncias em Hz para o eixo X e melhorar um pouco o entendimento do grfico anterior:

Para "corrigir" o eixo de magnitudes do espectro, devemos considerar que o algoritmo de FFT, escalona o vetor gerado, de um fator de (quantidade de amostras usadas). Ento inicialmente temos que redimencionar o vetor fazendo a diviso por : . E depois temos que considerar o espectro "refletido" gerado pelo algoritmo FFT. Na realidade este espectro refletido composto por conjugados complexos, e o espectro originalmente bilateral (vetor abaixo). E ainda deve ser considerado que o primeiro elemento calculado no espectro, corresponde frequncia nula (0 Hz, nvel DC do sinal).

Considerando os fatores anteriores, podemos gerar um novo grfico do espectro do sinal de entrada , fazendo:

xxxxxxxxxx
>> L=length(y);
>> Y=fft(y);
>> P2=abs(Y/L);                     % re-escalona magnitudes pelo fator L
>> P1 = P2(1:L/2+1);                % estamos interessados apenas na primeira metade do espectro
>> P1(2:end-1) = 2*P1(2:end-1);     % convertendo magnitudes para espectro unilateral
>> f = fs/L*(0:(L/2));              % monta vetor das frequencias
>> figure; plot(f,P1,"LineWidth",3) 
>> title('Espectro do sinal y(t)')
>> xlabel("freq. (Hz)")
>> ylabel('|fft(y)|')

O clculo anterior poderia ser simplificado para:

xxxxxxxxxx
>> L/2
ans =
   750
>> f = (fs/L)*(0:L-1);  
>> figure; plot(f, Y_mag/(L/2));

Que gera o mesmo grfico anterior, mas com o grfico ainda mostrando todo o espectro bilateral gerado originalmente pela funo FFT:

Calculando informao da Fase do sinal

Podemos ainda separar a informao de fase espectral do sinal de entrada:

>> Y_phase = angle(Y);
>> f = (fs/L)*(0:L-1);
>> figure; plot(f,Y_phase)
>> xlabel("freq. (Hz)")
>> ylabel('\angle (rad)')
>> title('Fase do Espectro do sinal y(t)')
>> axis([0 749 -pi pi])

Resumindo

A sequencia final de comandos que leva ao espectro de Magnitude do sinal de entrada pode ser feito da seguinte forma:

xxxxxxxxxx
>> Y=fft(y);
>> L=length(Y);
>> P2 = abs(Y/L);
>> P1 = P2(1:L/2+1);
>> P1(2:end-1) = 2*P1(2:end-1);
>> f = fs/L*(0:(L/2));
>> figure;
>> plot(f,P1,"LineWidth",3)
>> title('Espectro do sinal y(t)')
>> xlabel("freq. (Hz)")
>> ylabel('|fft(y)|')
>> % aplicando um "zoom" na regio at 100 Hz:
>> axis([0 100 0 3.5]) % mostra at freq de 100 Hz
